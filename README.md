# ionic-app
Mobile app usando cordova e ionic/Mobile cordova + ionic app

## Dependencias/Dependencies
npm
## Como rodar/How can i run it?

- npm install cordova -g
- npm install ionic -g
- android: 
    emulator:
        cordova run android --emulator
    device:
        cordova run android 
- web:
    ionic serve
